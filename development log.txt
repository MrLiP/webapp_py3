DAY 1 搭建开发环境

	安装开发所需第三方库
	异步框架aiohttp
	前端模板引擎jinja2
	MySQL 5.x数据库
	MySQL的Python异步驱动程序aiomysql
	
	搭建项目结构，即工作目录
	
		awesome-python3-webapp/  <-- 根目录
		|
		+- backup/               <-- 备份目录
		|
		+- conf/                 <-- 配置文件
		|
		+- dist/                 <-- 打包目录
		|
		+- www/                  <-- Web目录，存放.py文件
		|  |
		|  +- static/            <-- 存放静态文件
		|  |
		|  +- templates/         <-- 存放模板文件
		|
		+- ios/                  <-- 存放iOS App工程
		|
		+- LICENSE               <-- 代码LICENSE
	
	同步建立 git 仓库，并远程同步至 Github，保证代码修改的安全。
